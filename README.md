# 🚀 AppNexus - The Ultimate App Management Platform

**AppNexus** is a clean and responsive client-side web application designed for browsing and managing app data. Built as a part of a frontend, it allows users to explore a collection of apps, search in real-time, and simulate an installation experience using local storage.

---

## 🔗 Live Demo & Source
- **Live Site:** [View Live Demo](https://app-nexus-sable.vercel.app/)


## 🛠️ Technologies Used

- **React.js**: Core library for building the UI components.
- **Tailwind CSS**: For a minimalist, utility-first, and responsive design.
- **daisyUI**: Tailwind CSS components for clean and pre-styled UI elements.
- **React Router Dom (v6)**: For smooth navigation and dynamic routing.
- **React Toastify**: For beautiful, non-blocking success/error notifications.
- **Recharts**: For visualizing app review data through interactive charts.


## ✨ Key Features

### 1. 📱 All Apps & Live Search
- **Live Filtering:** Case-insensitive search bar that filters apps instantly as you type.
- **Dynamic Sort:** Sort apps by download count (High-to-Low and Low-to-High).


### 2. 📊 Detailed App Analytics
- **App Insights:** Comprehensive view of app metadata (Reviews, Downloads, Ratings).
- **Interactive Chart:** A responsive bar chart using **Recharts** to visualize star-rating distribution.
- **Description Slicing:** Dynamically formatted description.

### 4. 🏗️ Layout & UX
- **Active Navigation:** Visual indicators for current routes in the Header.
- **Loading Animations:** Smooth transitions during page navigation and search operations.
- **Error Handling:** A custom 404 Error Page and "No App Found" feedback.
